# NeuThesis是什么？
NeuThesis为 <b>N</b>orheastern<b></b> <b>U</b>niversity <b>Thesis</b> LaTeX Template之缩写，是ThuThesis的forked版本。

此宏包旨在建立一个简单易用的东北大学学位论文LaTeX模板，包括本科综合论文训练、硕士论文、博士论文、博士哲学论文以及博士后出站报告。现在支持本科、硕士、博士论文、博士后出站报告格式。
